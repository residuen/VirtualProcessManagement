OpenEdu-VirtualProcessManagement (version v 0.5 beta) by residuen


Das Programm

OpenEdu-VirtualProcessManagement ist ein OpenSource-Tool, welches die Simulation einer automatisierten Anlage ermoeglicht. Dies kann z.B. ein Lager- oder ein Warenhaus mit Wareneingang und Ausgang sein, oder aber auch eine Fertigungsstraße mit Robotern.
Derzeit besteht die Moeglichkeit, Hallen mittels eine CSV-Datei zu generieren, welche Wege fuer Maschinen und Menschen, Stapler, einfache Fahrzeuge, Ladung bzw. Ware und Lagerflaechen beinhalten.

Der HTTP-Server

Eine einfache HTTP-Anfrage an den Server veranlasst z.B. Bewegungen oder Lade-Entladevorgänge:
http://192.168.1.2/client?chargeobjectbyid=43,51,left z.B. veranlasst den Stapler mit der ID 43 Objekt 51 aufzuladen.
Der in Java geschriebene Server laeuft entweder auf dem lokalen Rechner, oder man startet den Server auf einem Server-PC auf den dann verschiedene Clients Zugriff haben.

Intention

Ziel ist das Planen und Entwickeln von simulierten Steuerungs- bzw. Warenmanagementanlagen. Aufgaben koennen sowohl selbststaendig, als auch in Teams in kooperativen Projektaufgaben durchgefuehrt werden.
Ein wesentlicher Vorteil bei der Implementierung per HTTP-Server ist, das die Loesung von Aufgaben Programmiersprachenunabhaengig ist. Auch die Anbindung einer Datenbank für eine Lagerverwaltung ist als Aufgabe moeglich, sowie die Eigenentwicklung einer textbasierten oder auch grafischen Benutzeroberflaeche fuer Steuerungs- und/oder Lagerlogistikaufgaben.
