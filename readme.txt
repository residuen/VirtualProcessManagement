VirtualProcessManagement
========================

Was ist es?
-----------
OpenEdu-VirtualProcessManagement ist ein OpenSource-Tool, welches die Simulation einer automatisierten Anlage ermoeglicht.
Dies kann z.B. ein Lager- oder ein Warenhaus mit Wareneingang und Ausgang sein, oder aber auch eine Fertigungsstraﬂe mit Robotern.

Mehr Details sind auf der Projektseite zu finden: http://residuen.github.com/VirtualProcessManagement/

Installation
------------
Um das Tool zu benutzen, muss mindestens Java 1.6 oder hoeher installiert sein.
Ein ausfuehrbares Jar-File liegt im GitHub bereit
